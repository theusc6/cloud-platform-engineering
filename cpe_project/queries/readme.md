
# Queries

This folder contains various queries used in the DevSecOps process for different purposes, such as infrastructure management and Identity or Access Management (IAM). The queries are organized into subfolders based on their categories.

## Table of Contents

- [Infrastructure Queries](#infrastructure-queries)
- [IAM Queries](#iam-queries)

### Infrastructure Queries

The `infrastructure` folder contains queries related to infrastructure management, such as resource information from S3 buckets, EC2 instances, etc.

#### IAM Queries

The `iam` folder contains queries focused on Identity and Access Management (IAM). These queries run reports on users, policies, roles, etc.

Please refer to each subfolder's README.md for more detailed information and usage instructions.
