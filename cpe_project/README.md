## Description

This repository is organized into various directories, each containing scripts, documentation, and resources for specific tasks related to cloud infrastructure management, GitHub automation, and more.

## Directory Descriptions

- [ACM](ACM/README.md): Scripts and documentation related to ACM (AWS Certificate Manager) management.
- [CONTRIBUTING.md](CONTRIBUTING.md): The requirement for contributing to this repository.
- [GENERAL.md](GENERAL.md): The requirement for general setup and guidlines on how to congigure advanced features within the repository.
- [config](config/README.md): Scripts and documentation for managing AWS Config and compliance-related tasks.
- [deployments](deployments/README.md): AWS Inspector scripts for deploying various components.
- [github](github/README.md): GitHub-related scripts and tools for repository and user management.
- [githubauthlib](github/githubauthlib/README.md): A Python library for GitHub authentication.
- [images](images/): The images used in the repository subdirectory.
- [infra](infra/README.md): Infrastructure-related scripts for services like API Gateway, DynamoDB, EC2, ELB, S3, and VPC.
- [outputs](outputs/README.md): Executed script outputs where applicable.
- [queries](queries/readme.md): Scripts for querying various AWS resources.
- [securityhub](securityhub/README.md): Scripts for managing AWS Security Hub and related tasks.
- [tests](tests/README.md): Unit tests for the scripts in the repository.
- [tools](tools/): Generic tools scripts not otherwise classified with greater specificity.
- [webapp](webapp/README.md): A web application related to AWS S3 bucket compliance.
